# Vibe: Playful / Agency / Experimental

**Vibe:** Awwwards winners, Stripe press, Vercel, Dribbble featured. Individual, avant-garde.

| Aspect | Characteristics |
|-------|--------------|
| **Layout** | Unconventional grid. Overlap. Scroll pinning. Split. Extreme asymmetric |
| **Color** | Bold. Saturated accent. Can use 2 accents. Bold gradient. Dark mode preferred |
| **Font** | Extreme display sans. Variable font weight extreme. Can use mono in body |
| **Corner radius** | Varied. Some sharp, some pill. Intentional |
| **Border** | Heavy `border-2` or none. Nothing in between |
| **Motion** | Cinematic. GSAP ScrollTrigger, physics, magnetic hover, parallax |
| **Typography** | Oversize font scale. Tracking extreme. Letter-spacing play |
| **Reference** | stripe.com, vercel.com, codrops.com, awwwards.com |

**Dials:** VARIANCE: 9-10 | MOTION: 8-10 | DENSITY: 3-4
